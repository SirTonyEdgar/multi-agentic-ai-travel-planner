URUTAN MENGHIDUPKAN SERVER (jalankan satu per satu, beda terminal):

------------------------------------------------------------
STEP 1 — MOCK API (port 8002)
------------------------------------------------------------
cd backend/mock-api
venv\Scripts\activate
uvicorn main:app --port 8002

------------------------------------------------------------
STEP 2 — INTEGRATION SERVICE (port 8000)
------------------------------------------------------------
cd backend/integration-service
venv\Scripts\activate
uvicorn main:app --port 8000

------------------------------------------------------------
STEP 3 — ORCHESTRATOR SERVICE (port 8001)
------------------------------------------------------------
cd backend/orchestrator-service
venv\Scripts\activate
uvicorn main:app --port 8001

------------------------------------------------------------
STEP 4 — API GATEWAY (port 8080)
------------------------------------------------------------
cd backend/api-gateway
venv\Scripts\activate
uvicorn main:app --port 8080

============================================================
  INSTALL DEPENDENCIES (sekali saja per service)

cd backend/integration-service
pip install fastapi uvicorn httpx

cd backend/orchestrator-service

============================================================
  CATATAN

- Urutan wajib: 8002 -> 8000 -> 8001 -> 8080

  isi: GOOGLE_API_KEY=isi_api_key_kamu

- Port summary:
  8002 = Mock API
  8000 = Integration Service
  8001 = Orchestrator Service
  8080 = API Gateway
